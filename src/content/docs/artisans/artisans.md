:::note
Artisan 開発を検討している場合は、まず要件を整理してから着手することを強く推奨します。
:::

Artisan API は、コンポジション内の 3D レイヤー描画を After Effects 標準レンダラーから置き換えるための API です。  
2D レイヤーの描画は引き続き After Effects 側が担当します。

Artisan はコンポジション単位で 1 つだけ選択でき、*コンポジション設定* > *詳細* から指定します。  
Artisan は 3D シーン全体をレンダリングし、必要情報をホストへ問い合わせながら処理するため、実装コストは高めです。

そのため、After Effects の既定 3D レンダリングを明確に置き換える必要がある場合に限定して採用してください。

Artisan はエフェクトと情報共有できますが、通信開始は Artisan 側主導です。  
多くの Artisan スイートは「全エフェクト適用後に生成されるレンダーコンテキスト」を前提とします。

---

## インタラクティブ Artisan

インタラクティブ Artisan は、最終レンダーではなくプレビュー表示専用に呼び出されます。  
最終出力時は既定 Artisan へフォールスルーします。

また、ユーザーが 3D 化したレイヤーだけでなく、コンポジション内の全レイヤーを対象に処理する点が通常 Artisan と異なります。
