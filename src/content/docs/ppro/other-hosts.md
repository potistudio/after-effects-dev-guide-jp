サードパーティホストの場合、Adobe のポリシーはそのまま残ります。

:::note
アドビでは、アドビと互換性のあるサードパーティ製ホストの作成をサポートまたは推奨しません。プラグイン API 仕様からのリバース エンジニアリングによって、部分的に機能するホストを作成できる場合がありますが、それはお勧めしません。また、そうすることについてはサポートされません。

:::

## リアリティサンドイッチ

私たちは、あなたのような開発者にとって、市場を成長させる良い方法の 1 つは、プラグインができるだけ多くのホストで動作するようにすることであると認識しています。

当社の SmartFX API により、After Effects API とプラグイン ホスティングの他の世界で利用可能な実装の間にかなりの距離が生まれました。

私たちは他のホストが新しい機能をサポートできるようできる限りのことを行います。サードパーティのホストで問題が発生した場合、サポートが必要な場合は、サードパーティのホストに問い合わせてください。
